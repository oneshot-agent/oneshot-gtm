# Built-with-oneshot-gtm badge

If you ship something using `oneshot-gtm`, drop this in your README. Saves us a sentence in the launch posts.

## Markdown

```markdown
[![Built with oneshot-gtm](https://img.shields.io/badge/built%20with-oneshot--gtm-0a0a0a?style=flat&labelColor=18181b&color=22c55e)](https://github.com/oneshot-agent/oneshot-gtm)
```

Renders as:

[![Built with oneshot-gtm](https://img.shields.io/badge/built%20with-oneshot--gtm-0a0a0a?style=flat&labelColor=18181b&color=22c55e)](https://github.com/oneshot-agent/oneshot-gtm)

## HTML

```html
<a href="https://github.com/oneshot-agent/oneshot-gtm">
  <img
    alt="Built with oneshot-gtm"
    src="https://img.shields.io/badge/built%20with-oneshot--gtm-0a0a0a?style=flat&labelColor=18181b&color=22c55e"
  />
</a>
```

## Variants

| Use case                              | Badge                                               |
| ------------------------------------- | --------------------------------------------------- |
| Default                               | `style=flat`                                        |
| Flat-square                           | `style=flat-square`                                 |
| For-the-badge                         | `style=for-the-badge`                               |
| With logo (custom svg via shields.io) | append `&logo=data:image/svg%2Bxml;base64,<base64>` |

The repo also embeds the same badge in its own README.
